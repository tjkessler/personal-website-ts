# Security Policy

## Supported versions

This repository is the source for [traviskessler.com](https://www.traviskessler.com). Security fixes are applied to the current `main` branch only.

## Reporting a vulnerability

Please report security issues privately. Do **not** open a public GitHub issue for vulnerabilities.

Prefer one of:

1. **GitHub private vulnerability reporting** — use [Report a vulnerability](https://github.com/tjkessler/personal-website-ts/security/advisories/new) on this repository (Security → Advisories), if enabled.
2. **Email** — [travis.j.kessler@gmail.com](mailto:travis.j.kessler@gmail.com) with a short description, steps to reproduce, and impact if known.

You should receive an acknowledgment within a few days. If the report is accepted, a fix will be prioritized for `main` and disclosed after a patch is available when appropriate.

## Scope

In scope: issues in this site’s source code, build/deploy configuration, or dependencies that could affect confidentiality, integrity, or availability of the deployed site.

Out of scope: content accuracy of the personal CV/portfolio, social-engineering against personal accounts, or third-party services linked from the site (unless the issue is clearly caused by this repository).
